# NQ Taxi - Frontend

This directory contains the frontend application for NQ Taxi.

## Tech Stack
- React.js / Next.js (to be initialized)
- HTML / CSS / JavaScript

## Getting Started

```bash
# Navigate to frontend directory
cd frontend

# Install dependencies
npm install

# Start development server
npm run dev
```
